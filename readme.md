
# Pomodoro Timer

Una aplicación de técnica Pomodoro construida con React Native.

## Características

- ⏱️ Temporizador de 25 minutos para trabajo
- ☕ Descansos de 5 minutos
- 🔔 Notificaciones al completar ciclos
- 📊 Seguimiento de sesiones completadas
- 🎨 Interfaz limpia e intuitiva

## Instalación

```bash
npm install
# o
yarn install
```

## Uso

```bash
npm start
# o
yarn start
```

## Requisitos

- Node.js 14+
- React Native CLI
- Android Studio o Xcode (según tu plataforma)

## Estructura del Proyecto

```
src/
    components/     # Componentes reutilizables
    screens/        # Pantallas principales
    utils/          # Funciones auxiliares
    App.js          # Archivo principal
```

## Tecnologías

- React Native
- JavaScript/TypeScript
